# Readme
For local development only

## Installation

Run
```bash
docker-compose up -d
```

Down
```bash
docker-compose down
```

Access PhpMyAdmin
```bash
localhost:8081
```

Port:
```bash
Mysql: 4037
PhpMyAdmin: 8082
```

Credentials:
```bash
mysql root username: root
mysql root password: secret
```
